meia query = media type + media features.

meia type = tipo da media

midia features = caracteristica da media

quando eu utilizar media query, posso usar ela pra vários formatos de tela.
media types posso escrever normal, porém media features, tenho que escrever entre parênteses.

exemplos:
	meda type = screen, impressoras
	media features = screen (orientation: landscape).
